# IP Changer for DSM
- IP를 변경할 일이 많은 대마인들이 IP를 쉽게 변경할 수 있다.
- 2019-03-26 V.1


## How
- using C++, MFC


## Version.1
![MAIN_UI](./docs/img/MAIN_UI.png)
- Change your IP, SubnetMask, Default Gateway, DNS, SubDNS(static/DHCP)

- 자신만의 IP를 저장 / 로드 할 수 있는 기능

- 입력된 IP와 DNS를 한꺼번에 지우는 버튼
